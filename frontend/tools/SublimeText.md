#SublimeText

快捷键
------

Ctrl+K+B 隐藏展示侧边栏

Alt+L 需自行配置，隐藏展示行号

F12 GoToAnywhere

F11 全屏

配置
----

```
{
    "font_size": 12,
    "ignored_packages":
    [
        "Markdown",
        "Vintage"
    ],
    "intent_to_bracket": true,
    "line_numbers": true,
    "rulers":
    [
        120
    ],
    "tab_size": 4,
    "translate_tabs_to_spaces": true
}
```
